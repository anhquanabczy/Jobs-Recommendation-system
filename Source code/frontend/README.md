# Job Recommender System 

Hệ thống gợi ý việc làm sử dụng (Ensemble Learning, BERT) và ReactJS.

## 1. Cài đặt Backend
cd backend
python -m venv venv
# Windows:
.\venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

pip install -r requirements.txt
python main.py

## 2. Cài đặt Frontend
cd frontend
npm install
npm run dev
